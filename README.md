## Deployment instructions

### Init

```
> npm install
> firebase signin
```

### Serve locally

```
> npm start
```

### Deploy

```
# first build the project
> npm run build

# deploy to dev
> npm run deploy_dev

# deploy to bridgestars.net
> npm run deploy
```
